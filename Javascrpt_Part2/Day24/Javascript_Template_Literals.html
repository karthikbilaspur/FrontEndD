JavaScript Template Literals

Template literals are string literals that allow embedded expressions (variables) into your code.

Basic Example

let a='GFG'
console.log(`hello ${a}`)

Output: hello GFG

String Interpolation with Expressions

let x = 5;
let y = 10;
console.log(`The sum of ${x} and ${y} is ${x + y}`);

Output: The sum of 5 and 10 is 15

Multi-line Strings with Template Literals

const s = `This is a multi-line
string that spans across
several lines.`;
console.log(s);

Output: This is a multi-line
string that spans across
several lines.

Tagged Template Literals

function greet(strings, name) {
    return `${strings[0]}${name.toUpperCase()}${strings[1]}`;
}
const name = 'gfg';
console.log(greet`Hello, ${name}!`);

Output: Hello, GFG!

Escaping Backticks and Dollar Signs

const s = `This is a backtick: \` and this is a dollar sign: \$`;
console.log(s);

Output: This is a backtick: ` and this is a dollar sign: $

Use Cases

  1. Multi-line Strings

    const poem = `Roses are red,
Violets are blue,
JavaScript is awesome,
And so are you!`;
console.log(poem);

    Output: Roses are red,
    Violets are blue,
    JavaScript is awesome,
    And so are you!

  2. Dynamic Expressions

    const a = 5, b = 10;
const result = `Sum of ${a} and ${b} is ${a + b}.`;
console.log(result);

    Output: Sum of 5 and 10 is 15.

  3. Tagged Templates

    function tag(strings, ...values) {
    return strings.reduce((acc, str, i) => acc + str + (values[i] || ''), '');
}
const output = tag`Hello, ${"Saran"}!`;
console.log(output);

    Output: Hello, Saran!

  4. HTML Template

    const title = "Welcome";
const html = `
    ${title}
    `;
console.log(html);

    Output:

    Welcome

  5. Conditionals in Templates

    const isAdmin = true;
const userRole = `User role: ${isAdmin ? "Admin" : "Guest"}.`;
console.log(userRole);

    Output: User role: Admin.

  6. Loops with Templates

    const items = ["apple", "banana", "cherry"];
const list = `Items: ${items.map(item => `\n- ${item}`)}`;
console.log(list);

    Output: Items:
    - apple,
    - banana,
    - cherry

  7. Embedding Functions

    const toUpper = str => str.toUpperCase();
const s = `Shouting: ${toUpper("hello")}`;
console.log(s);

    Output: Shouting: HELLO

Hello World Example

// hello.ts
function greet(name: string) {
console.log(Hello, ${name}!);
}
greet("John");
Code

Saving the File

Save this file with a .ts extension, e.g., hello.ts.

Compiling the File

To compile the TypeScript file, run the following command in your terminal:

tsc hello.ts

This will generate a hello.js file that you can run with Node.js:

node hello.js

TypeScript Configuration

Create a tsconfig.json file to configure the TypeScript compiler:

{
"compilerOptions": {
    "target": "es6",
    "module": "commonjs",
    "outDir": "build"
},
"include": ["src/**/*"]
}

This configuration tells TypeScript to:

  • Target ES6 syntax
  • Use CommonJS modules
  • Output compiled files to the build directory
  • Include files in the src directory and its subdirectories

Running the Compiler with Configuration

To compile the TypeScript files using the tsconfig.json configuration, run:

tsc

This will compile all files in the src directory and output the compiled JavaScript files to the build directory.

Example with Classes and Interfaces

// person.ts
interface Person {
name: string;
age: number;
}
class Employee implements Person {
name: string;
age: number;
employeeId: number;
Code
constructor(name: string, age: number, employeeId: number) {
    this.name = name;
    this.age = age;
    this.employeeId = employeeId;
}

greet() {
    console.log(`Hello, my name is ${this.name} and I'm ${this.age} years old.`);
}
}
const emp = new Employee("John Doe", 30, 123);
emp.greet();

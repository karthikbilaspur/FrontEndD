ReactJS useNavigate Hook

The useNavigate hook is a part of the React Router library, which allows you to navigate between routes programmatically.

What is useNavigate?

useNavigate is a hook that returns a function that can be used to navigate to a new location.

Example Usage:

import React from 'react';
import { useNavigate } from 'react-router-dom';
Code
  const Home = () => {
    const navigate = useNavigate();

    const handleClick = () => {
      navigate('/about');
    };

    return (
      <div>
        <h1>Home</h1>
        <button onClick={handleClick}>Go to About</button>
      </div>
    );
  };

How it works:

  • The useNavigate hook returns a navigate function.
  • The navigate function takes a URL path as an argument.
  • When called, it navigates to the specified URL.

Options:

  • navigate('/about'): Navigate to the specified URL.
  • navigate('/about', { replace: true }): Replace the current URL in the history stack.
  • navigate(-1): Go back to the previous URL.

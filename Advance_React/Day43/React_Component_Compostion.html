React Component Composition

Component composition is a powerful technique in React that allows you to break down complex UI components into smaller, reusable pieces.

When to use Component Composition?

  • When you have a complex UI component that can be broken down into smaller, reusable pieces.
  • When you want to reuse a component with different content or styles.
  • When you want to manage state and props more efficiently.

How to use Component Composition?

There are several techniques for component composition in React, including:

  • Containment (Children Props): Used for building reusable wrapper components.
  • Specialization (Props-based Composition): Create specialized versions of components by passing props.
  • Higher-Order Components (HOCs): Functions that take a component as input and return an enhanced component.
  • Render Props: Pass a function as a prop to give more control over rendering.

Example

Let's take a look at an example of component composition using Containment (Children Props):

// Card.js
const Card = ({ children, style }) => {
  return (
    <div style={{ ...style, border: '1px solid #ddd', padding: 20, borderRadius: 10 }}>
      {children}
    </div>
  );
};

export default Card;
    
// App.js
import Card from './Card.js';

const App = () => {
  return (
    <div>
      <Card style={{ backgroundColor: 'lightblue' }}>
        <h2>Card Title</h2>
        <p>This is a card component.</p>
      </Card>
      <Card style={{ backgroundColor: 'lightgreen' }}>
        <h2>Another Card</h2>
        <p>This is another card component.</p>
      </Card>
    </div>
  );
};

export default App;

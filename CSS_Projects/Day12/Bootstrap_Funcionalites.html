Bootstrap

Bootstrap is a popular front-end framework for building responsive, mobile-first web applications.

Major Functionalities

1. Grid System

Bootstrap provides a responsive grid system for building layouts.

<div class="container"> <div class="row"> <div class="col-md-6">Column 1</div> <div class="col-md-6">Column 2</div> </div> </div>
Column 1
Column 2

2. UI Components

Bootstrap provides a wide range of pre-built UI components, such as buttons, forms, and navigation.

<button type="button" class="btn btn-primary">Primary</button>

3. Responsive Utilities

Bootstrap provides responsive utility classes for hiding or displaying content based on screen size.

<div class="d-none d-md-block">Visible on medium and larger screens</div>
Visible on medium and larger screens

4. Customization

Bootstrap allows customization through Sass variables and mixins.

// Customizing the primary color $primary: #007bff;

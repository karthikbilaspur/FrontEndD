CSS Frameworks

CSS frameworks are pre-written CSS code libraries that make it easier to style and layout web pages.

Types of CSS Frameworks

1. Utility-First Frameworks

These frameworks provide low-level utility classes to style individual elements.

<button class="bg-blue-500 hover:bg-blue-700 text-white font-bold py-2 px-4 rounded"> Click me </button>

Example: Tailwind CSS

2. Component-Based Frameworks

These frameworks provide pre-built UI components, such as buttons, forms, and navigation.

<button type="button" class="btn btn-primary">Primary</button>

Example: Bootstrap

3. Grid-Based Frameworks

These frameworks provide a grid system for building responsive layouts.

<div class="grid-container"> <div class="grid-x"> <div class="cell small-6">Cell 1</div> <div class="cell small-6">Cell 2</div> </div> </div>

Example: Foundation

4. Hybrid Frameworks

These frameworks combine elements of utility-first, component-based, and grid-based frameworks.

<div class="columns"> <div class="column is-6">Column 1</div> <div class="column is-6">Column 2</div> </div>

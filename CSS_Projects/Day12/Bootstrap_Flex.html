Bootstrap Flex

Bootstrap provides a powerful flexbox system for building responsive layouts.

Enable Flexbox

To enable flexbox, add the .d-flex class to the parent element.

<div class="d-flex"> <div>Item 1</div> <div>Item 2</div> <div>Item 3</div> </div>
Item 1
Item 2
Item 3

Flex Direction

Use the .flex-row class to set the flex direction to row (default).

<div class="d-flex flex-row"> <div>Item 1</div> <div>Item 2</div> <div>Item 3</div> </div>
Item 1
Item 2
Item 3

Use the .flex-column class to set the flex direction to column.

<div class="d-flex flex-column"> <div>Item 1</div> <div>Item 2</div> <div>Item 3</div> </div>
Item 1
Item 2
Item 3

Justify Content

Use the .justify-content-start, .justify-content-end, .justify-content-center, .justify-content-between, and .justify-content-around classes to justify the content.

<div class="d-flex justify-content-start"> <div>Item 1</div> <div>Item 2</div> <div>Item 3</div> </div> <div class="d-flex justify-content-end"> <div>Item 1</div> <div>Item 2</div> <div>Item 3</div> </div> <div class="d-flex justify-content-center"> <div>Item 1</div> <div>Item 2</div> <div>Item 3</div> </div> <div class="d-flex justify-content-between"> <div>Item 1</div> <div>Item 2</div> <div>Item 3</div> </div> <div class="d-flex justify-content-around"> <div>Item 1</div> <div>Item 2</div> <div>Item 3</div> </div>

Align Items

Use the .align-items-start, .align-items-end, .align-items-center, .align-items-baseline, and .align-items-stretch classes to align the items.

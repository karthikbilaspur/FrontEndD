Tailwind CSS Layout

Tailwind CSS is a utility-first CSS framework for rapid custom UI.

Container

Fix the max-width of an element to match the min-width of the breakpoint.

<div class="container mx-auto"> Container </div>
Container

Box Sizing

Define how the user should calculate the total width and height of an element.

<div class="box-border h-32 w-32 p-4 border-4 border-green-500"> Box Sizing </div>
Box Sizing

Display

Define how the components are going to be placed on the web page.

<div class="block p-4 bg-blue-500 text-white"> Block </div> <div class="inline p-4 bg-blue-500 text-white"> Inline </div>
Block
Inline

Float

Define the flow of content for controlling the wrapping of content around an element.

<div class="float-right p-4 bg-red-500 text-white"> Float Right </div>
Float Right

Clear

Specify which side of floating elements are not allowed to float.

<div class="clear-both p-4 bg-gray-200"> Clear Both </div>
Clear Both

Object Fit

Specify how an image or video should be resized to fit its content box.

<img class="object-cover h-32 w-32" src="image.jpg">

Object Position

Specify how an image or video element is positioned with x/y coordinates.

<img class="object-none object-left-top h-32 w-32" src="image.jpg">

Overflow

Tell whether to clip content or to add scroll bars.

<div class="overflow-auto h-32 w-32 p-4 bg-gray-200"> Overflow Auto </div>
Overflow Auto

Overscroll Behavior

Set the behavior of the browser when the boundary of a scrolling area is reached.

<div class="overscroll-contain h-32 w-32 p-4 bg-gray-200"> Overscroll Contain </div>
Overscroll Contain

Position

Control how an element is positioned in the DOM.

<div class="absolute top-0 left-0 p-4 bg-blue-500 text-white"> Absolute </div>
Absolute

Top/Right/Bottom/Left

Control the alignment of a positioned element.

Top Right

Invisible

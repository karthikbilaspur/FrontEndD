CSS Background Properties

Property Description
background-color Sets the background color of an element.
background-image Sets the background image of an element.
background-repeat Specifies how the background image should be repeated.
background-position Sets the position of the background image.
background-attachment Specifies whether the background image should be fixed or scroll with the page.
background A shorthand property for setting all background properties at once.

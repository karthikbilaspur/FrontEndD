Selectors

  • Universal Selector: * { margin: 0; padding: 0; }
  • Type Selector: h1 { font-family: Arial, sans-serif; }
  • Class Selector: .box { border: 1px solid black; padding: 10px; }
  • ID Selector: #header { background-color: lightgray; }

Declaration Block

  • Properties: color, background-color, font-size, margin, padding
  • Values: color name, number (e.g., 16px), percentages (e.g., 50%)

Grouping and Nesting

  • Grouping: h1, h2, h3 { color: darkblue; }
  • Nesting: ul li { list-style-type: square; }

Pseudo-classes and Pseudo-elements

  • Pseudo-class: a:hover { color: green; }
  • Pseudo-element: p::first-line { font-weight: bold; }

CSS Flexbox Properties

Property Description
display Specifies the display type (e.g., flex, inline-flex).
flex-direction Specifies the direction of the flex items (e.g., row, column).
flex-wrap Specifies whether the flex items should wrap or not (e.g., wrap, nowrap).
justify-content Specifies the alignment of the flex items along the main axis (e.g., flex-start, center, space-between).
align-items Specifies the alignment of the flex items along the cross axis (e.g., flex-start, center, baseline).

CSS Font Properties

Property Description
font-family Specifies the font family for text.
font-size Specifies the font size of text.
font-style Specifies the font style (e.g., italic, oblique).
font-weight Specifies the font weight (e.g., bold, normal).
font-variant Specifies the font variant (e.g., small-caps).
font A shorthand property for setting font properties.
text-align Specifies the alignment of text.
text-decoration Specifies the decoration of text (e.g., underline).
text-transform Specifies the transformation of text (e.g., uppercase).

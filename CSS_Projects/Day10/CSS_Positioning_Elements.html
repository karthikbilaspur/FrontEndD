CSS Positioning Elements

CSS positioning elements are used to control the layout of HTML elements on a web page.

Types of Positioning Elements

  • Static: This is the default positioning element. Elements are positioned according to the normal flow of the document.
  • Relative: Elements are positioned relative to their normal position.
  • Absolute: Elements are positioned relative to their nearest positioned ancestor.
  • Fixed: Elements are positioned relative to the viewport.
  • Sticky: Elements are positioned based on the user's scroll position.

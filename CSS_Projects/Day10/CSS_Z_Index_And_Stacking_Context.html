Rules for Stacking Context

  • The stacking context is formed by an element that is positioned and has a z-index other than auto.
  • The stacking context is also formed by an element that is a flex item or grid item.
  • The stacking context is also formed by an element that has an opacity less than 1.
  • The stacking context is also formed by an element that has a transform, filter, or clip-path property.

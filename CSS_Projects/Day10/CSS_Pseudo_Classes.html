Pseudo Classes

Pseudo classes are used to define a special state of an element.

Types of Pseudo Classes

  • :hover - selects an element when it is hovered over
  • :active - selects an element when it is being activated (e.g. clicked)
  • :focus - selects an element when it has focus (e.g. a form input)
  • :visited - selects a link when it has been visited
  • :first-child - selects an element that is the first child of its parent
  • :last-child - selects an element that is the last child of its parent

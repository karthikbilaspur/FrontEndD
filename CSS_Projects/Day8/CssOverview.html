CSS (Cascading Style Sheets)

What is CSS?

CSS stands for Cascading Style Sheets. It is a styling language used to control the layout and appearance of web pages written in HTML or XML.

Types of CSS

  • Inline CSS: Applied directly to an HTML element using the style attribute.
  • Internal CSS: Defined in the head section of an HTML document using the style tag.
  • External CSS: Defined in an external file (usually with a .css extension) and linked to an HTML document.

Uses of CSS

  • Layout and spacing: CSS is used to control the layout and spacing of web pages, including margins, padding, and positioning.
  • Typography: CSS is used to control the typography of web pages, including font family, size, color, and style.
  • Color and background: CSS is used to control the color and background of web pages, including background images and gradients.
  • Responsive design: CSS is used to create responsive designs that adapt to different screen sizes and devices.
  • Animations and transitions: CSS is used to create animations and transitions that enhance the user experience.

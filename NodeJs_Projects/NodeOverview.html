What Can You Build With Node.js?

  • ✔️ Real-time apps (like chat)
  • ✔️ High-traffic websites
  • ✔️ Web servers and websites
  • ✔️ REST APIs
  • ✔️ Command-line tools
  • ✔️ IoT and hardware control
  • ✔️ Working with files and databases

Key Features:

  • ✔️ Event-driven, non-blocking model
  • ✔️ Handles many connections at once

How to Run Node.js Code:

  • 1. Save code in a file (e.g., app.js)
  • 2. Run with node app.js

Node.js Built-in Modules:

  • ✔️ Use require() to import modules (e.g., os, http)
  • Examples: os, http, fs, path

npm (Node Package Manager):

  • ✔️ Install and manage third-party packages (libraries)
  • Example: npm install express

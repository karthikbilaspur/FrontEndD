What are Async Functions?

Async functions in Node.js are used to handle asynchronous operations, allowing your code to run without blocking other tasks.

Key Features:

  • ✔️ async keyword defines an asynchronous function
  • ✔️ await keyword pauses execution until a promise is resolved
  • ✔️ Returns a Promise

Functionality in Node.js:

  • ✔️ Handle I/O operations (e.g., reading files, network requests)
  • ✔️ Improve code readability and maintainability
  • ✔️ Avoid callback hell

Example:


const fs = require('fs').promises;

async function readFile() {
  try {
    const data = await fs.readFile('example.txt', 'utf8');
    console.log(data);
  } catch (err) {
    console.error(err);
  }
}

readFile();

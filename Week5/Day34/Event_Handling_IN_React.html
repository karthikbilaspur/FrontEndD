In React, event handling is used to handle user interactions, such as clicks, form submissions, and keyboard inputs.

Basic Event Handling

To handle events in React, you can pass a function as a prop to a component.


import React from 'react';

function handleClick() {
  console.log('Button clicked!');
}

function MyButton() {
  return <button onClick={handleClick}>Click me!</button>;
}

Passing Arguments to Event Handlers

You can pass arguments to event handlers using arrow functions or the bind method.


import React from 'react';

function handleClick(id) {
  console.log(id);
}

function MyButton({ id }) {
  return <button onClick={() => handleClick(id)}>Click me!</button>;
}

Event Object

The event object passed to event handlers is a synthetic event, with properties like target, currentTarget, preventDefault(), and stopPropagation().


import React from 'react';

function handleClick(event) {
  console.log(event.target.value);
}

function MyInput() {
  return <input type="text" onClick={handleClick} />;
}

Common Event Handling Patterns

  • Use event.persist() when accessing event properties asynchronously.
  • Memoize event handlers with useCallback to prevent unnecessary re-renders.
  • Use arrow functions to pass arguments to event handlers.

React Lifecycle

Overview

The React lifecycle refers to the series of methods that are called at different stages of a component's life, from its creation to its removal from the DOM.

Phases of the React Lifecycle

  • Mounting: The component is being created and added to the DOM.
  • Updating: The component is being updated due to changes in props or state.
  • Unmounting: The component is being removed from the DOM.
  • Error Handling: The component encounters an error.

Mounting Phase

  • constructor(): Initializes the component's state and props.
  • static getDerivedStateFromProps(): Updates the state based on props changes.
  • render(): Returns the JSX that defines the component's UI.
  • componentDidMount(): Called after the component is rendered to the DOM.

Updating Phase

  • static getDerivedStateFromProps(): Updates the state based on props changes.
  • shouldComponentUpdate(): Determines whether the component should update.
  • render(): Returns the JSX that defines the component's UI.
  • getSnapshotBeforeUpdate(): Captures a snapshot of the component's state before updating.
  • componentDidUpdate(): Called after the component is updated.

Unmounting Phase

  • componentWillUnmount(): Called before the component is removed from the DOM.

Error Handling Phase

  • static getDerivedStateFromError(): Updates the state when an error occurs.
  • componentDidCatch(): Catches errors in the component tree.

Example Use Cases

Example 1: Logging Lifecycle Methods


import React, { Component } from 'react';

class Logger extends Component {
  constructor(props) {
    super(props);
    console.log('Constructor called');
  }

  static getDerivedStateFromProps() {
    console.log('getDerivedStateFromProps called');
    return null;
  }

  componentDidMount() {
    console.log('componentDidMount called');
  }

  shouldComponentUpdate() {
    console.log('shouldComponentUpdate called');
    return true;
  }

  getSnapshotBeforeUpdate() {
    console.log('getSnapshotBeforeUpdate called');
    return null;
  }

  componentDidUpdate() {
    console.log('componentDidUpdate called');
  }

  componentWillUnmount() {
    console.log('componentWillUnmount called');
  }

  render() {
    console.log('render called');
    return <div>Logger Component</div>;
  }
}

export default Logger;

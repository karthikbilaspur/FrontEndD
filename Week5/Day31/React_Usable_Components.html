How to make Reusable React Components?

React components are modular pieces of UI that take data via props to render dynamic and complex interfaces.

Key Factors for Reusable Components

  1. Avoid side effects: Don't include logic that interacts with external data directly inside a reusable component.
  2. Use Props: Pass data and logic as props to customize the component's behavior and appearance.

Example: Bad Practice

const ButtonReusable = () => {
  return <button> Click Me </button>;
}

Example: Good Practice

const ButtonReusable = ({ color, text }) => {
  return <button style={{ backgroundColor: color }}>{text}</button>;
}

Optimizing Performance

Use memoization to avoid unnecessary re-renders. React's React.memo and useMemo hooks can help.

Example: Reusable Components

// src/App.js
import React from 'react';
import ProductList from './ProductList.js';

const App = () => {
  const products = [
    { id: 1, name: 'Product 1', price: 10 },
    { id: 2, name: 'Product 2', price: 20 },
    { id: 3, name: 'Product 3', price: 30 },
  ];

  return (
    <div style={{ margin: '5px' }}>
      <h2 style={{ color: 'green' }}>
        Reusable Components Example
      </h2>
      <ProductList products={products} />
    </div>
  );
};

export default App;

React Components

What are React Components?

Components are the building blocks of React apps. They're reusable pieces of UI that manage their own state and props.

Example:

function Button() {
return <button>Click me!</button>;
}

Types of Components

  • Functional Components (recommended): 
    function Button() {
return <button>Click!</button>;
}
  • Class Components (legacy): 
    class Button extends React.Component {
render() {
return <button>Click!</button>;
}
}

Component Best Practices

  • Single Responsibility: One component, one job.
  • Reusability: Make components flexible with props.
  • Naming: Use PascalCase (e.g., UserProfile).
  • Props Validation: Use propTypes (optional).
  • State: Use useState hook in functional components.
  • Keep it simple: Avoid complex logic in components.

Example: Functional Component with Props

function Greeting({ name }) {
return <h1>Hello, {name}!</h1>;
}
// Usage:
<Greeting name="John" />

Example: Component with State

function Counter() {
const [count, setCount] = useState(0);
return (
<div>
<p>Count: {count}</p>
<button onClick={() => setCount(count + 1)}>+</button>
</div>
);
}

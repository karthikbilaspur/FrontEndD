Installing React

Prerequisites

  • Node.js (LTS version recommended)
  • npm (comes with Node.js) or yarn
  • A code editor like VS Code

Creating a New React App

Use create-react-app (CRA) for a quick setup:

npx create-react-app my-react-app
cd my-react-app
npm start

This sets up a basic React project with:

  • Webpack for bundling
  • Babel for transpiling
  • ESLint for linting

Choosing Frameworks & Tools

Popular choices:

  • State Management: Redux, MobX, or React Context
  • Routing: React Router
  • Styling: CSS Modules, Styled Components, or Sass
  • Testing: Jest + React Testing Library

Example with React Router:

npm install react-router-dom

Manual Setup (Advanced)

If you don't want CRA:

npm init -y
npm install react react-dom
npm install webpack webpack-cli babel-loader --save-dev

Configure webpack.config.js and .babelrc manually.

React JSX

What is JSX?

JSX (JavaScript XML) is a syntax extension for JavaScript used in React. It allows writing HTML-like code in JavaScript files.

const element = <h1>Hello, World!</h1>;

How JSX Works

  • JSX is converted to React.createElement() calls by Babel.
  • Creates React elements, which are plain objects describing the UI.
// JSX
const element = <h1>Hello</h1>;
// Compiled to
const element = React.createElement('h1', null, 'Hello');

Implementation

  • Use JSX in React components to define UI.
  • Wrap expressions in {} to evaluate JavaScript.
function App() {
const name = 'React';
return <h1>Hello, {name}!</h1>;
}

JSX Attributes

  • Use camelCase for attributes (e.g., className, onClick).
  • Pass props like HTML attributes.
function Button() {
return <button className="btn" onClick={() => alert('Clicked')}>Click</button>;
}

JSX Objects

JSX elements are treated as objects:

const element = {
type: 'h1',
props: {
children: 'Hello'
}
};

Useful for conditional rendering:

const isAdmin = true;
const element = isAdmin ? <h1>Welcome Admin</h1> : <p>Not authorized</p>;

Key Points

  • JSX is optional but recommended for React.
  • Gets compiled to React.createElement().
  • Use {} for JavaScript expressions.

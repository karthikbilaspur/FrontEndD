React Functional Components

Overview

Functional components are a type of React component that uses a function to render JSX. They're a simpler way to write components, and are often preferred over class components for their conciseness and ease of use.

Syntax


import React from 'react';

function MyComponent(props) {
  return <div>Hello, World!</div>;
}

export default MyComponent;

Alternatively, you can use an arrow function:


import React from 'react';

const MyComponent = (props) => {
  return <div>Hello, World!</div>;
};

export default MyComponent;

Key Features

  • No this keyword: Functional components don't have their own this context.
  • No lifecycle methods: Functional components don't have lifecycle methods like componentDidMount. Instead, use Hooks like useEffect.
  • State with Hooks: Use Hooks like useState to manage state in functional components.

Using Props


import React from 'react';

function Greeting(props) {
  return <div>Hello, {props.name}!</div>;
}

export default Greeting;

You can also destructure props:


import React from 'react';

function Greeting({ name }) {
  return <div>Hello, {name}!</div>;
}

export default Greeting;

Using Hooks


import React, { useState } from 'react';

function Counter() {
  const [count, setCount] = useState(0);

  return (
    <div>
      <p>Count: {count}</p>
      <button onClick={() => setCount(count + 1)}>Increment</button>
    </div>
  );
}

export default Counter;

Benefits

  • Simpler code: Functional components are often more concise than class components.
  • Easier to test: Functional components are easier to test because they don't have a complex lifecycle.
  • Better performance: Functional components can be optimized by React for better performance.

Example Use Cases

Example 1: Simple Greeting Component


import React from 'react';

function Greeting({ name }) {
  return <div>Hello, {name}!</div>;
}

export default Greeting;

Example 2: Todo List Component


import React, { useState } from 'react';

function TodoList() {
  const [todos, setTodos] = useState([
    { id: 1, text: 'Buy milk' },
    { id: 2, text: 'Walk the dog' },
  ]);

  return (
    <ul>
      {todos.map((todo) => (
        <li key={todo.id}>{todo.text}</li>
      ))}
    </ul>
  );
}

export default TodoList;

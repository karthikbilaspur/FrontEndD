HTML Table

An HTML table is a way to display data in a tabular format using rows and columns.

<table> <tr> <th>Header 1</th> <th>Header 2</th> </tr> <tr> <td>Data 1</td> <td>Data 2</td> </tr> </table>

Related Tags

  • <table>: defines a table
  • <tr>: defines a table row
  • <th>: defines a table header cell
  • <td>: defines a table data cell
  • <caption>: defines a table caption

Example 1: Simple Table

<table> <tr> <th>Name</th> <th>Age</th> </tr> <tr> <td>John</td> <td>25</td> </tr> <tr> <td>Jane</td> <td>30</td> </tr> </table>

Example 2: Table with Caption and Colspan

<table> <caption>Employee Information</caption> <tr> <th>Name</th> <th>Age</th> <th>Department</th> </tr> <tr> <td>John</td> <td>25</td> <td rowspan="2">Sales</td> </tr> <tr> <td>Jane</td> <td>30</td> </tr> </table>

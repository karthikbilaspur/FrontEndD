HTML List

An HTML list is a way to display a collection of items in a formatted way.

Code

Types of HTML Lists

  • Ordered List (OL): <ol> - used for numbered lists
  • Unordered List (UL): <ul> - used for bulleted lists
  • Description List (DL): <dl> - used for lists with descriptions

Related Tags

  • <li>: defines a list item
  • <dt>: defines a term in a description list
  • <dd>: defines a description in a description list

Example 1: Ordered List

<ol> <li>Item 1</li> <li>Item 2</li> <li>Item 3</li> </ol>

Example 2: Unordered List

<ul> <li>Item A</li> <li>Item B</li> <li>Item C</li> </ul>

Example 3: Description List

<dl> <dt>Term 1</dt> <dd>Description of Term 1</dd> <dt>Term 2</dt> <dd>Description of Term 2</dd> </dl>

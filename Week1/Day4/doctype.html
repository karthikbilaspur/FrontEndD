What is DOCTYPE?

The DOCTYPE declaration is the first line of code in an HTML document, indicating the document type and version of HTML being used.

Why is it necessary?

It is necessary for:

  • Browser Rendering Mode: Triggers the browser to render the document in standards mode.
  • Validation: Helps validators and editors understand the document structure.
  • Compatibility: Ensures cross-browser compatibility and proper rendering.

In HTML5, the DOCTYPE declaration is simplified to <!DOCTYPE html>.

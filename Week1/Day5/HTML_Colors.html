HTML Colors

What are HTML Colors?

HTML colors are used to specify the color of text, backgrounds, borders, and other elements in an HTML document.

Color Formats

There are several ways to specify colors in HTML:

  • Color Name: A color name, such as "red", "green", or "blue".
  • HEX Code: A hexadecimal code, such as "#FF0000" for red.
  • RGB Value: An RGB value, such as "rgb(255, 0, 0)" for red.
  • RGBA Value: An RGBA value, such as "rgba(255, 0, 0, 0.5)" for red with 50% opacity.

Examples:

Color Names:

color: red;
color: green;
color: blue;

HEX Codes:

color: #FF0000;
color: #00FF00;
color: #0000FF;

RGB Values:

color: rgb(255, 0, 0);
color: rgb(0, 255, 0);
color: rgb(0, 0, 255);

RGBA Values:

color: rgba(255, 0, 0, 0.5);
color: rgba(0, 255, 0, 0.5);
color: rgba(0, 0, 255, 0.5);

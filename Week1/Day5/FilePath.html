File Paths

What are File Paths?

A file path is a string of characters that specifies the location of a file or directory in a computer's file system.

Types of File Paths

There are two main types of file paths:

  • Absolute Path: An absolute path is a complete path from the root directory to the file or directory.
  • Relative Path: A relative path is a path relative to the current working directory.

Examples:

Absolute Paths:

C:\Users\username\Documents\file.txt (Windows) /home/username/Documents/file.txt (Unix/Linux)

Relative Paths:

file.txt (current directory) ./file.txt (current directory) ../file.txt (parent directory) ./dir/file.txt (subdirectory)

File Path Notation

  • . : Current directory
  • .. : Parent directory
  • / : Root directory (Unix/Linux)
  • \ : Directory separator (Windows)

Best Practices

  • Use relative paths whenever possible
  • Use forward slashes (/) as directory separators
  • Avoid using absolute paths
  • Use lowercase letters for file names and directories

HTML Entities

What are HTML Entities?

HTML entities are used to represent special characters in HTML, such as <, >, &, and others.

Why Use HTML Entities?

HTML entities are used to prevent the browser from interpreting special characters as HTML code, and to ensure that the characters are displayed correctly.

Common HTML Entities

  • <: < (less than)
  • >: > (greater than)
  • &: & (ampersand)
  • ": " (double quote)
  • ': ' (single quote)
  •  : (non-breaking space)
  • ©: (copyright symbol)
  • ®: (registered trademark symbol)
  • ™: (trademark symbol)

Entity Types

  • Named Entity: &entity_name; (e.g., <)
  • Numeric Entity: &#entity_number; (e.g., <)
  • Hexadecimal Entity: ntity_hex; (e.g., <)

Examples

  • <p>This is a paragraph</p>:

    This is a paragraph

  • &copy; 2023: © 2023
  • "Hello, World!": "Hello, World!"

Block and Inline Elements in HTML5

What are Block Elements?

Block elements are HTML elements that occupy the full width available, and they start on a new line. They have a rectangular shape and can contain other block-level elements or inline elements.

What are Inline Elements?

Inline elements, on the other hand, occupy only the space needed for their content and do not start on a new line. They are typically used within block-level elements.

Examples:

Block Elements:

<div>This is a block element</div>
<p>This is a paragraph, also a block element</p>
<h1>This is a heading, a block element too</h1>

Inline Elements:

<span>This is an inline element</span>
<a href="#">This is a link, an inline element</a>
<strong>This text is strong, an inline element</strong>

Here's a simple HTML5 page demonstrating block and inline elements:

<!DOCTYPE html>
<html>
<head>
    <title>Block and Inline Elements</title>
    <style>
        div, p, h1 {
            border: 1px solid black;
            margin: 10px;
        }
        span, a, strong {
            background-color: lightblue;
        }
    </style>
</head>
<body>
    <h1>Block Elements</h1>
    <div>This is a block element</div>
    <p>This is a paragraph, also a block element</p>
    <h1>This is a heading, a block element too</h1>

    <h1>Inline Elements</h1>
    <p>This is a <span>inline element</span> within a paragraph.</p>
    <p>This is a <a href="#">link, an inline element</a> within a paragraph.</p>
    <p>This text is <strong>strong, an inline element</strong> within a paragraph.</p>
</body>
</html>

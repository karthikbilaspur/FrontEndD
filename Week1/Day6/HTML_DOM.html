What is HTML DOM?

The HTML Document Object Model (DOM) is a programming interface for HTML and XML documents. It represents the structure of a document as a tree of objects, allowing developers to interact with and manipulate the document's content and layout.

Types of HTML DOM

  • Core DOM: The core DOM is the base model for all types of documents.
  • XML DOM: The XML DOM is used for XML documents.
  • HTML DOM: The HTML DOM is used for HTML documents.

Examples

1. Accessing an Element by ID

// Get the element with the ID "myElement"
var element = document.getElementById("myElement");
console.log(element);

2. Changing an Element's Text Content

// Get the element with the ID "myElement"
var element = document.getElementById("myElement");
// Change the text content of the element
element.textContent = "New text content!";

3. Adding an Event Listener

// Get the element with the ID "myButton"
var button = document.getElementById("myButton");
// Add an event listener to the button
button.addEventListener("click", function() {
console.log("Button clicked!");
});

4. Creating a New Element

// Create a new paragraph element
var paragraph = document.createElement("p");
// Set the text content of the paragraph
paragraph.textContent = "This is a new paragraph!";
// Add the paragraph to the document body
document.body.appendChild(paragraph);

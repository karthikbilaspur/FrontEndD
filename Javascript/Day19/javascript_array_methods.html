Array Methods in JavaScript

JavaScript provides several array methods that can be used to manipulate and iterate over arrays.

Mutator Methods

These methods modify the original array.

1. push()

const arr = [1, 2, 3];
arr.push(4, 5);
console.log(arr); // [1, 2, 3, 4, 5]

2. pop()

const arr = [1, 2, 3, 4, 5];
arr.pop();
console.log(arr); // [1, 2, 3, 4]

3. shift()

const arr = [1, 2, 3, 4, 5];
arr.shift();
console.log(arr); // [2, 3, 4, 5]

4. unshift()

const arr = [2, 3, 4, 5];
arr.unshift(1);
console.log(arr); // [1, 2, 3, 4, 5]

5. splice()

const arr = [1, 2, 3, 4, 5];
arr.splice(1, 2);
console.log(arr); // [1, 4, 5]

Accessor Methods

These methods do not modify the original array.

1. concat()

const arr1 = [1, 2, 3];
const arr2 = [4, 5, 6];
const arr3 = arr1.concat(arr2);
console.log(arr3); // [1, 2, 3, 4, 5, 6]

2. slice()

const arr = [1, 2, 3, 4, 5];
const arr2 = arr.slice(1, 3);
console.log(arr2); // [2, 3]

3. join()

const arr = [1, 2, 3, 4, 5];
const str = arr.join(', ');
console.log(str); // "1, 2, 3, 4, 5"

Iteration Methods

These methods iterate over the array and perform a specified action.

1. forEach()

const arr = [1, 2, 3, 4, 5];
arr.forEach((num) => console.log(num));

2. map()

const arr = [1, 2, 3, 4, 5];
const arr2 = arr.map((num) => num * 2);
console.log(arr2); // [2, 4, 6, 8, 10]

3. filter()

const arr = [1, 2, 3, 4, 5];
const arr2 = arr.filter((num) => num % 2 === 0);
console.log(arr2); // [2, 4]

4. reduce()

const arr = [1, 2, 3, 4, 5];
const sum = arr.reduce((acc, num) => acc + num, 0);
console.log(sum); // 15

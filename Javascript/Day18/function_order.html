Higher-Order Functions

A higher-order function is a function that takes another function as an argument or returns a function as its result.

Why Higher-Order Functions?

  • They allow us to abstract away complex logic and make our code more reusable.
  • They enable us to create functions that can be customized with different behaviors.
  • They are a key feature of functional programming.

Examples

Simple Higher-Order Function

function twice(func) {
func();
func();
}
twice(() => console.log('Hello World!'));

Higher-Order Function with Arguments

function greet(greeting) {
return function(name) {
console.log(`${greeting}, ${name}!`);
}
}
const sayHello = greet('Hello');
sayHello('John'); // Output: Hello, John!

Array Methods as Higher-Order Functions

Array Method map()

const numbers = [1, 2, 3, 4, 5];
const doubleNumbers = numbers.map((num) => num * 2);
console.log(doubleNumbers); // Output: [2, 4, 6, 8, 10]

Array Method filter()

const numbers = [1, 2, 3, 4, 5];
const evenNumbers = numbers.filter((num) => num % 2 === 0);
console.log(evenNumbers); // Output: [2, 4]

Array Method reduce()

const numbers = [1, 2, 3, 4, 5];
const sum = numbers.reduce((acc, num) => acc + num, 0);
console.log(sum); // Output: 15

Array Method forEach()

const numbers = [1, 2, 3, 4, 5];
numbers.forEach((num) => console.log(num));

Array Method find()

const numbers = [1, 2, 3, 4, 5];
const found = numbers.find((num) => num > 3);
console.log(found); // Output: 4

Array Method some()

const numbers = [1, 2, 3, 4, 5];
const hasEven = numbers.some((num) => num % 2 === 0);
console.log(hasEven); // Output: true

Array Method every()

const numbers = [1, 2, 3, 4, 5];
const allEven = numbers.every((num) => num % 2 === 0);
console.log(allEven); // Output: false

Function Composition

Function Composition

const addOne = (x) => x + 1;
const multiplyByTwo = (x) => x * 2;
const composed = (x) => multiplyByTwo(addOne(x));
console.log(composed(5)); // Output: 12

Currying

Currying

function add(a) {
return function(b) {
return a + b;
}
}
const addFive = add(5);
console.log(addFive(3)); // Output: 8

Memoization

Memoization

function memoize(func) {
const cache = {};
return function(...args) {
const key = JSON.stringify(args);
if (cache[key]) {
return cache[key];
}
const result = func(...args);
cache[key] = result;
return result;
}
}
const add = (a, b) => a + b;
const memoizedAdd = memoize(add);
console.log(memoizedAdd(2, 3)); // Output: 5

Passing Functions as Arguments

Passing Functions as Arguments

function greet(func) {
func();
}
greet(() => console.log('Hello World!'));

Returning Functions from Functions

Returning Functions from Functions

function greet() {
return function() {
console.log('Hello World!');
}
}
const sayHello = greet();
sayHello();

JavaScript If Statement

The if statement is used to execute a block of code if a specified condition is true.

Types of If Statements

Type Description Example
If Statement Executes a block of code if the condition is true. if (x > 5) { console.log("x is greater than 5"); }
If-Else Statement Executes a block of code if the condition is true, otherwise executes another block of code. if (x > 5) { console.log("x is greater than 5"); } else { console.log("x is less than or equal to 5"); }
If-Else If-Else Statement Executes a block of code if the condition is true, otherwise checks another condition and executes a block of code if true, otherwise executes another block of code. if (x > 5) { console.log("x is greater than 5"); } else if (x === 5) { console.log("x is equal to 5"); } else { console.log("x is less than 5"); }
Nested If Statement An if statement inside another if statement. if (x > 5) { if (y > 10) { console.log("x is greater than 5 and y is greater than 10"); } }

Examples

If Statement

let x = 10;
if (x > 5) {
console.log("x is greater than 5");
}

If-Else Statement

let x = 3;
if (x > 5) {
console.log("x is greater than 5");
} else {
console.log("x is less than or equal to 5");
}

If-Else If-Else Statement

let x = 5;
if (x > 5) {
console.log("x is greater than 5");
} else if (x === 5) {
console.log("x is equal to 5");
} else {
console.log("x is less than 5");
}

Nested If Statement

let x = 10;
let y = 15;
if (x > 5) {
if (y > 10) {
console.log("x is greater than 5 and y is greater than 10");
}
}

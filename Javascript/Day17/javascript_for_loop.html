JavaScript For Loop

The for loop is used to execute a block of code repeatedly for a specified number of times.

Types of For Loops

Type Description Example
For Loop Executes a block of code repeatedly for a specified number of times. for (let i = 0; i < 5; i++) { console.log(i); }
For-In Loop Executes a block of code for each property of an object. for (let prop in obj) { console.log(prop); }
For-Of Loop Executes a block of code for each element of an iterable (array, string, etc.). for (let elem of arr) { console.log(elem); }

Examples

For Loop

for (let i = 0; i < 5; i++) {
console.log(i);
}
// Output: 0, 1, 2, 3, 4

For-In Loop

let obj = { a: 1, b: 2, c: 3 };
for (let prop in obj) {
console.log(prop + ": " + obj[prop]);
}
// Output: a: 1, b: 2, c: 3

For-Of Loop

let arr = [1, 2, 3];
for (let elem of arr) {
console.log(elem);
}
// Output: 1, 2, 3

For Loop with Break Statement

for (let i = 0; i < 5; i++) {
if (i === 3) {
break;
}
console.log(i);
}
// Output: 0, 1, 2

For Loop with Continue Statement

for (let i = 0; i < 5; i++) {
if (i === 3) {
continue;
}
console.log(i);
}
// Output: 0, 1, 2, 4

JavaScript Control Flow Statements

Control flow statements are used to control the flow of execution of a program.

Types of Control Flow Statements

Type Description Example
Break Statement Terminates the current loop or switch statement. for (let i = 0; i < 5; i++) { if (i === 3) { break; } console.log(i); }
Continue Statement Skips the current iteration of a loop. for (let i = 0; i < 5; i++) { if (i === 3) { continue; } console.log(i); }
Return Statement Returns a value from a function. function add(a, b) { return a + b; }
Throw Statement Throws an exception. throw new Error("Error message");

Examples

Break Statement

for (let i = 0; i < 5; i++) {
if (i === 3) {
break;
}
console.log(i);
}
// Output: 0, 1, 2

Continue Statement

for (let i = 0; i < 5; i++) {
if (i === 3) {
continue;
}
console.log(i);
}
// Output: 0, 1, 2, 4

Return Statement

function add(a, b) {
return a + b;
}
console.log(add(2, 3));
// Output: 5

Throw Statement

try {
throw new Error("Error message");
} catch (e) {
console.log(e.message);
}
// Output: Error message

Try-Catch Block

try {
// Code that may throw an error
} catch (e) {
// Code to handle the error
} finally {
// Code to be executed regardless of an error
}

Switch Statement

let day = "Monday";
switch (day) {
case "Monday":
console.log("Today is Monday");
break;
case "Tuesday":
console.log("Today is Tuesday");
break;
default:
console.log("Today is not Monday or Tuesday");
}
// Output: Today is Monday

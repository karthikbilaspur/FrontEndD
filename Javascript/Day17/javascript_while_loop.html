JavaScript While Loop

The while loop is used to execute a block of code repeatedly as long as a specified condition is true.

Types of While Loops

Type Description Example
While Loop Executes a block of code repeatedly as long as the condition is true. let i = 0; while (i < 5) { console.log(i); i++; }
Do-While Loop Executes a block of code once, then repeatedly as long as the condition is true. let i = 0; do { console.log(i); i++; } while (i < 5);

Examples

While Loop

let i = 0;
while (i < 5) {
console.log(i);
i++;
}
// Output: 0, 1, 2, 3, 4

Do-While Loop

let i = 0;
do {
console.log(i);
i++;
} while (i < 5);
// Output: 0, 1, 2, 3, 4

While Loop with Break Statement

let i = 0;
while (i < 5) {
if (i === 3) {
break;
}
console.log(i);
i++;
}
// Output: 0, 1, 2

While Loop with Continue Statement

let i = 0;
while (i < 5) {
i++;
if (i === 3) {
continue;
}
console.log(i);
}
// Output: 1, 2, 4, 5

Infinite While Loop

let i = 0;
while (true) {
console.log(i);
i++;
if (i === 5) {
break;
}
}
// Output: 0, 1, 2, 3, 4

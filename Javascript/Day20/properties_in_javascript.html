JavaScript Object Properties

Object properties are the values associated with an object, identified by a key (string or symbol).

Property Types

  • Data Properties: Contain a value
  • Accessor Properties: Define getter/setter functions

Property Characteristics

Characteristic Description
Value The property's value
Writable Whether the value can be changed
Enumerable Whether it appears in for...in loops
Configurable Whether it can be deleted or modified

Defining Properties

1. Object Literal

const person = {
  name: "John",
  age: 30
};

2. Dot Notation

const person = {};
person.name = "John";
person.age = 30;

3. Bracket Notation

const person = {};
person["name"] = "John";
person["age"] = 30;

4. Object.defineProperty()

const person = {};
Object.defineProperty(person, 'name', {
  value: "John",
  writable: true,
  enumerable: true
});

Property Access

const person = { name: "John", age: 30 };
console.log(person.name); // John
console.log(person["age"]); // 30
Note: Property names are strings or symbols. If you use other types, they're converted to strings.

Iterating Over JavaScript Objects

There are several ways to iterate over object properties in JavaScript.

Methods to Iterate

Method Description Includes Prototypes
for...in Iterates over all enumerable properties Yes
Object.keys() Returns array of own enumerable keys No
Object.values() Returns array of own enumerable values No
Object.entries() Returns array of own enumerable key-value pairs No

Examples

1. for...in Loop

const person = {name: "John", age: 30};
for (let key in person) {
  console.log(`${key}: ${person[key]}`);
}

Output: name: John, age: 30

2. Object.keys()

const person = {name: "John", age: 30};
Object.keys(person).forEach(key => {
  console.log(`${key}: ${person[key]}`);
});

Output: name: John, age: 30

3. Object.entries()

const person = {name: "John", age: 30};
for (let [key, value] of Object.entries(person)) {
  console.log(`${key}: ${value}`);
}

Output: name: John, age: 30

Note: Use hasOwnProperty() with for...in to exclude prototype properties if needed.

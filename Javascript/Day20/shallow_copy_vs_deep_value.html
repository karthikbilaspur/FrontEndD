Deep Copy vs Shallow Copy in JavaScript

Shallow Copy

A shallow copy creates a new object and copies the references of the original object's properties. Changes to nested objects affect both the original and the copy.

const original = { a: 1, b: { c: 2 } };
const shallowCopy = { ...original };
shallowCopy.b.c = 3;
console.log(original.b.c); // 3

Deep Copy

A deep copy creates a new object and recursively copies the values of the original object's properties. Changes to nested objects do not affect the original.

const original = { a: 1, b: { c: 2 } };
const deepCopy = JSON.parse(JSON.stringify(original));
deepCopy.b.c = 3;
console.log(original.b.c); // 2

Methods for Deep Copy

  • JSON.parse(JSON.stringify(obj)) - Simple but has limitations (e.g., doesn't handle functions, undefined, or circular references)
  • structuredClone(obj) - Modern approach, handles more cases but not supported in older browsers
  • Libraries like Lodash (_.cloneDeep())

Comparison

Feature Shallow Copy Deep Copy
Nested Objects Copies reference Copies value
Performance Faster Slower
Independence Nested objects are linked Fully independent
Note: Choose shallow copy for simple objects, deep copy for complex nested structures.

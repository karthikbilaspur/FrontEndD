JavaScript Objects

Objects are the fundamental data structure in JavaScript, used to store collections of key-value pairs.

Creating Objects

1. Object Literal Syntax

const person = {
  name: "John Doe",
  age: 30,
  profession: "Developer"
};

Output: { name: 'John Doe', age: 30, profession: 'Developer' }

2. Using new Object()

const person = new Object();
person.name = "John Doe";
person.age = 30;
person.profession = "Developer";

Output: { name: 'John Doe', age: 30, profession: 'Developer' }

Object Operations

  • Access Properties: obj.name or obj["name"]
  • Modify: obj.age = 31;
  • Add: obj.city = "New York";
  • Remove: delete obj.profession;

Object Methods

  • Object.keys(obj) - Returns property names
  • Object.values(obj) - Returns property values
  • Object.entries(obj) - Returns key-value pairs

{} vs new Object()

Feature {} new Object()
Usage Preferred, concise Less common
Performance Faster Slower
Best Practice: Use object literals ({}) for creating objects. It's more concise and efficient.

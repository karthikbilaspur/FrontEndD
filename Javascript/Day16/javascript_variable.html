JavaScript Scope

Types of Scope

Scope Description Example
Global Scope Variables declared outside any function or block are accessible everywhere in the code. var x = 10; console.log(x); // 10
Function Scope Variables declared with var inside a function are accessible only within that function. function example() { var x = 10; console.log(x); // 10 }
Block Scope Variables declared with let or const inside a block ({}) are accessible only within that block. if (true) { let x = 10; console.log(x); // 10 }
Modular Scope Introduced in ES6, ES Modules are the standard way to organize JavaScript code into reusable and maintainable chunks. module.exports = { PI, add };

Examples

Global Scope

var x = 10;
console.log(x); // 10
let y = 20;
console.log(y); // 20
const z = 30;
console.log(z); // 30

Block Scope

if (true) {
var x = 10; // Function-scoped (not block-scoped)
let y = 20; // Block-scoped
const z = 30; // Block-scoped
}
console.log(x); // 10
console.log(y); // Error: y is not defined
console.log(z); // Error: z is not defined

Function Scope

function example() {
var x = 10; // Function-scoped
let y = 20; // Function-scoped
const z = 30; // Function-scoped
console.log(x, y, z); // 10 20 30
}
example();
console.log(x); // Error: x is not defined
console.log(y); // Error: y is not defined
console.log(z); // Error: z is not defined

Lexical Scope (Closures)

function outer() {
let outerVar = "I'm in the outer scope!";
function inner() {
console.log(outerVar); // Accessing parent's scope
}
inner();
}
outer();

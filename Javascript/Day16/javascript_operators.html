JavaScript Operators

Types of Operators

Operator Description Example
Arithmetic Operators Perform mathematical calculations like addition, subtraction, multiplication, etc. const sum = 5 + 3; // 8
Assignment Operators Assign values to variables and perform operations like addition or multiplication. let n = 10; n += 5; // 15
Comparison Operators Compare two values and return a boolean (true or false). console.log(10 > 5); // true
Logical Operators Perform logical operations like AND, OR, NOT. console.log(true && false); // false
Bitwise Operators Perform operations on binary representations of numbers. console.log(5 & 1); // 1
Ternary Operator A shorthand for conditional statements. const status = age >= 18 ? "Adult" : "Minor";
Comma Operator Evaluate multiple expressions and return the last value. const res = (n1 = 1, n2 = 2, n1 + n2); // 3
Unary Operators Operate on a single operand (e.g., increment, decrement). let x = 5; console.log(++x); // 6
Relational Operators Check relationships between values. console.log("length" in obj); // true
BigInt Operators Perform calculations with large integers. const big1 = 123n; console.log(big1 + 1n); // 124n
String Operators Concatenate strings. const s = "Hello" + " World"; // "Hello World"
Chaining Operator (?.) Safe navigation operator. obj.address?.city; // "Delhi"

Examples

Arithmetic Operators

const sum = 5 + 3; // 8
const diff = 10 - 2; // 8
const p = 4 * 2; // 8
const q = 8 / 2; // 4

Assignment Operators

let n = 10;
n += 5; // 15
n *= 2; // 30

Comparison Operators

console.log(10 > 5); // true
console.log(10 === "10"); // false

Logical Operators

const a = true, b = false;
console.log(a && b); // false
console.log(a || b); // true

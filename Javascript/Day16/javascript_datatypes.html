JavaScript Data Types

Primitive Types

Type Description Example
String Represents a sequence of characters. let name = 'John';
Number Represents both integers and floating-point numbers. let age = 30; let height = 5.9;
BigInt Represents integers with arbitrary precision. let bigNumber = 123456789012345678901234567890n;
Boolean Represents a logical value, either true or false. let isAdmin = true;
Null Represents the intentional absence of any object value. let data = null;
Undefined Represents an uninitialized variable or a non-existent property. let x; console.log(x); // undefined
Symbol Represents a unique identifier. let id = Symbol('id');

Reference Types

Type Description Example
Object Represents a collection of key-value pairs. let person = { name: 'John', age: 30 };
Array A special type of object representing a collection of elements. let numbers = [1, 2, 3, 4, 5];
Function A special type of object that can be invoked. function add(a, b) { return a + b; }
Date Represents a date and time. let now = new Date();
RegExp Represents a regular expression. let regex = /hello/;
